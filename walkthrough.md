# Autodrop UI Walkthrough

I have initialized the Next.js foundation and built out the Premium UI for both the **Landing Page** and the **Analytics Dashboard** as requested.

## Landing Page
The landing page features a sleek dark-mode aesthetic with ambient glassmorphism blobs, gradients, and high-quality typography (Inter). 
- It includes the primary hero section highlighting the value proposition.
- It displays a semi-transparent glass showcase panel previewing the dashboard.

## Analytics Dashboard
The dashboard (`/dashboard`) contains:
- **KPI Cards** (Active Automations, Comments Matched, DMs Delivered, Leads Captured)
- **Activity Streams** showing live trigger events like "DM Sent" and "Follow-Gate"
- **Sidebar Navigation** spanning Overview, Automations, Leads, Logs, and Settings
